# Network Packet Sniffer

A Python based network packet sniffer for real-time network traffic analysis and basic security monitoring.

## Features

- Live packet capture
- Source & Destination IP detection
- Protocol detection (TCP, UDP, ICMP, DNS)
- Port detection
- Packet length monitoring
- Traffic statistics
- Common service identification (HTTP, HTTPS, DNS, SSH)

## Technologies

- Python
- Scapy
- Networking
- Cybersecurity
